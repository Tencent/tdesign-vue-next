    何时使用 
    栅格通常用于内容区域的布局设计。
    推荐/慎用示例 
    在栅格中放置内容区块时，内容区块的位置应该从列开始，到列结束。 
    一行中不建议放置过多内容区块，内容区块数量建议控制在4个以内。 
    更多栅格使用可参考： 全局样式-Layout布局
