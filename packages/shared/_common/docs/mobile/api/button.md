### 类型

#### 基础按钮

##### 次按钮

使用场景：在用户进行的操作为流程中的辅助操作，或者进行不那么重要的交互行为时，选择用次按钮；次要按钮通常和主要按钮一起出现

##### 主按钮

使用场景：大部分场景都可以使用，例如反馈页、表单页、对话框，一个页面建议最多只出现一个主按钮；

##### 文字按钮

使用场景：它的操作通常和其旁边内容相关，通常出现在标题旁、字段旁、列表最下方
