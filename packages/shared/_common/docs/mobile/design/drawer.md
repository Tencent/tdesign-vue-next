### 相似组件

| 组件名                      | 何时使用                                                                                     |
| :-------------------------- | :------------------------------------------------------------------------------------------- |
| [动作面板](./action-sheet)  | 需要提供一组与当前场景操作相关的关联操作时使用。                                             |
| [对话框](./dialog)          | 需要用户做一些决定，或这提供完成某个任务是需要的一些额外信息时使用。                         |
| [下拉菜单](./dropdown-menu) | 当内容较多时，需要通过筛选快速定位某一类内容时使用。                                         |
| [选择器](./dropdown-menu)   | 当需要在有限的空间展示大量选项供用户选择，或者一组选项由递进层级构成需要用户逐级选择时使用。 |
