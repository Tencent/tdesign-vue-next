

### 何时使用

当前流程的结束或新流程的开启需要用户点击触发时使用。

### 相似组件

| 组件名                 | 何时使用                                                    |
| :--------------------- | :---------------------------------------------------------- |
| [链接](./link)         | 当需要外链跳转或页面底部footer链接跳转时使用。              |
| [悬浮按钮](./fab)      | 当某个操作为全局操作，且为用户高频/业务强推的操作时可使用。 |
| [返回顶部](./back-top) | 当页面内容过长，用户有快速返回到页面顶部的诉求时使用。      |
