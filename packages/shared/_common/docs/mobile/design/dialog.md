### 何时使用
| 组件名                | 何时使用                                                                                   |
| :-------------------- | :----------------------------------------------------------------------------------------- |
| [轻提示](./toast)     | 当需要对用户进行轻量反馈或提示，且不需要用户进行操作时使用，轻提示会在一段时间后自动消失。 |
| [消息通知](./message) | 当需要对用户进行较轻量的反馈或提示，可以自动消失或通过点击关闭，通常由用户触发。           |
