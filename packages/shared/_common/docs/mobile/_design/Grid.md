# 宫格 Grid

## 组件设计指南

### 何时使用

当需要展示多个功能或信息入口，且这些入口没有明显的优先级时使用。

### 组件搭配使用

##### 可与徽标组合使用，可作为消息提示、或其它相关的数量提示。

### 推荐/慎用示例

##### 当在金刚区使用宫格承载功能入口时，建议梳理功能的优先级，将金刚区控制在3行以内，一些相对次要的功能可通过左右滑动的方式收纳在第二屏。
