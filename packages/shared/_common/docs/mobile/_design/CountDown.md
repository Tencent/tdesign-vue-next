# 倒计时 CountDown

## 组件设计指南

### 何时使用

当需要向用户提示剩余时间时使用。

### 常见用法

##### 在营销场景下，通过倒计时向用户展示运营活动开始倒计时或距活动结束的剩余时间，从而体现稀缺性吸引目标受众的注意力，勾起足够兴趣以此激励用户参与。
