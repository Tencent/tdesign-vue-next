### 相似组件

| 组件名                 | 何时使用                                                                           |
| :--------------------- | :--------------------------------------------------------------------------------- |
| [选择器](./select)     | 需要在有限的空间展示大量选项，并从中选取单个或多个选项时。用于收纳较多选项入口时。 |
| [下拉菜单](./dropdown) | 下拉菜单主要用于对过多的操作进行收纳，常跟随于按钮后面，点击后是触发相应的操作。   |
