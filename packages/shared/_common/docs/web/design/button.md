### 推荐/慎用示例

##### 用户在使用主、次按钮时，需要表达主次关系，突出强调的最主要操作，其他操作使用次按钮，禁止同时使用多个主按钮。

### 相似组件

| 组件名                  | 何时使用                                                     |
| :---------------------- | :----------------------------------------------------------- |
| [按钮](./button)        | 按钮用于开启一个闭环的操作任务，如“删除”对象、“购买”商品等。 |
| [菜单](./menu)          | 一系列按钮的集合，主要用于节省界面空间。                     |
| 文字链                  | 指向的是一个内部或外部的页面。点击后页面跳转至指向页。       |
