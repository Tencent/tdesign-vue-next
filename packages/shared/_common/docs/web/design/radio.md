### 何时使用

用于在多个互斥的备选项中选中一个选项时。

### 组件搭配使用

##### 与文字提示搭配使用，当单选框需要对某种状态进行说明时，利用文字提示进行解释说明。

### 相似组件

| 组件名                 | 何时使用                                                                                                     |
| :--------------------- | :----------------------------------------------------------------------------------------------------------- |
| [单选框](./radio)      | 需要从一组互斥的选项中仅选择一个选项时                                                                       |
| [下拉菜单](./dropdown) | 需要在有限的空间内，需要承载多种操作，并且不影响页面结构时；需要将不常用的操作进行收纳，减少对空间的占用时。 |
