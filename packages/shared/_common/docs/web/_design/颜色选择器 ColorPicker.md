# 颜色选择器 ColorPicker

## 组件设计指南

### 何时使用

当用户需要对形状、字体或可修改的元素自定义更改颜色时。


### 与页面布局相关

##### 固定布局：可布局于左侧或右侧编辑栏上，区分内容读取区和工具操作区，高效引导用户发现和使用。
