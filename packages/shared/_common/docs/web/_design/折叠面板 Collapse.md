# 折叠面板 Collapse

## 组件设计指南

### 何时使用

需要将分组内容收纳或隐藏时；

信息内容较多或较复杂，需引导用户先浏览主题，再查看详情内容时。

### 与布局相关

##### 通常布局在内容区，对不同类型的信息进行组织分类，便于用户检索和查看。

### 组件搭配使用
##### 与表单搭配使用，在展开的面板中进行信息的添加或编辑，可以给用户扁平沉浸的编辑状态，避免弹出层或进入二级界面带来的任务打断。
