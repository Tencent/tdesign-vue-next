# 选项卡Tabs
用于承载同一层级下不同页面或类别的组件，方便用户在同一个页面框架下进行快速切换。




### 基础选项卡
使用选项卡切换内容模块，操作后不会进行页面跳转。

![简约选项卡](https://tdesign.gtimg.com/site/design/images/简约选项卡-1820490.jpg)


### 带图标选项卡
在基础选项卡基础上，在每个标签前添加图标，方便用户快速理解。

![带icon选项卡](https://tdesign.gtimg.com/site/design/images/带icon选项卡-1820506.jpg)


### 增删选项卡
用户可添加、删除选项卡，满足自定义场景。

![带操作选项卡](https://tdesign.gtimg.com/site/design/images/带操作选项卡-1820552.jpg)
