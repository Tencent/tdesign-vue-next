### 水平步骤条
以水平方向引导用户使用的步骤条。

#### 带序号的水平步骤条
适用于步骤数较多时，让用户更明确的了解步骤数量。

{{ sequence }}

#### 不带序的水平步骤条

适用于步骤数较少时，主要引导用户按步骤完成操作。

{{ no-sequence }}

### 垂直步骤条

垂直方向排列的步骤条，常用于竖排布局或窄屏场景。
#### 不带序号的垂直步骤条

适用于垂直步骤较少的情况。

{{ vertical-no-sequence }}

#### 带序号的垂直步骤条
适用于步骤较多、步骤提示内容较长的场景。

{{ vertical-sequence }}
